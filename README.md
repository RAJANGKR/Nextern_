## 📌 Overview

**Nextern** is a full-stack, AI-augmented placement preparation platform built for engineering students. It centralises everything a student needs — structured topic tracking, real-time activity feeds, job drive discovery, AI-generated study roadmaps, resume analysis, and performance analytics — into a single, cohesive product.

---

## ✨ Features

### 🧠 Prep Hub — Structured Learning Tracker
Track your readiness across the five core Computer Science subjects evaluated in placement interviews.

| Subject | Topics Covered |
|---|---|
| **DSA** | Arrays, Linked Lists, Trees, Graphs, DP, Sorting, Searching, … |
| **DBMS** | Normalisation, SQL, Transactions, Indexing, … |
| **Operating Systems** | Processes, Scheduling, Memory Management, Deadlocks, … |
| **Computer Networks** | OSI Model, TCP/IP, DNS, HTTP, Sockets, … |
| **OOPs** | Inheritance, Polymorphism, Abstraction, Design Patterns, … |

- Sub-topic level checkbox tracking with live progress bars  
- Progress persisted per user in MongoDB and synced on every session  
- Visual readiness score for each subject

---

### 📰 Activity Feed
A peer-driven, role-aware feed to keep students informed and engaged.

- Post placement updates, resources, and experiences  
- Like and comment on posts  
- Role-based filtering (student / admin)  
- Automated announcement injection via scheduled scrapers  

---

### 🏢 Placement Drives
Centralised job drive discovery and application management.

- Aggregated drives from Adzuna, Google Jobs, Microsoft Careers, and more via a built-in web scraper  
- Filter by branch, graduation year, and eligibility criteria  
- One-click application with status tracking  
- Admin controls to publish, edit, and close drives  

---

### 🗺️ AI-Powered Roadmap Generator
Personalised study plans generated by Google Gemini.

- Input your target company, role, and timeline  
- Receive a week-by-week, topic-ordered preparation plan  
- Roadmap adapts to your existing Prep Hub progress  
- Plans saved to your profile for ongoing reference  

---

### 📄 AI Resume Analyzer
Upload your resume (PDF) and receive intelligent, recruiter-aligned feedback.

- Parses PDF content using `pdf-parse`  
- Analyses against industry standards via Gemini / Anthropic Claude  
- Highlights strengths, gaps, and actionable improvement suggestions  
- Scores the resume across key sections (skills, experience, projects, …)  

---

### 📊 Analytics Dashboard
Personal placement readiness metrics at a glance.

- Subject-wise progress charts  
- Application funnel tracking (applied → shortlisted → offered)  
- Activity streaks and engagement stats  
- Admin-level platform analytics (user growth, drive metrics, post activity)  

---

### 🔔 Notifications
- Real-time in-app notifications for drive deadlines, post interactions, and announcements  
- Notification centre with read/unread state management  

---

### 👤 Profile & Onboarding
A rich, recruiter-ready profile for every student.

- Personal, academic, and career information  
- Skills, certifications, and language proficiencies  
- Links to LinkedIn, GitHub, LeetCode, and portfolio  
- Target companies, preferred roles, and relocation preferences  
- Profile completeness gate before accessing the platform  

---

### 🛡️ Admin Panel
Dedicated controls for platform administrators.

- Manage users, drives, and posts  
- Publish announcements  
- Monitor platform analytics  
- Role-based access control with admin-only middleware  
- Admin action audit log

## 🚀 Getting Started

### Prerequisites

- **Node.js** v18 or higher
- **MongoDB** — local instance or [MongoDB Atlas](https://cloud.mongodb.com) (free tier)
- A **Google Cloud** project with OAuth 2.0 credentials
- A **Google AI Studio** API key (Gemini)

### 1 — Clone the Repository

```bash
git clone https://github.com/RAJANGKR/Nextern_.git
cd Nextern_
```

### 2 — Configure Environment Variables

Copy the example file and fill in your values:

```bash
cp server/.env.example server/.env
```

```env
# Server
PORT=4000
NODE_ENV=development

# Database
MONGO_URI=mongodb+srv://<user>:<password>@cluster.mongodb.net/nextern

# Authentication
JWT_SECRET=your_super_secret_jwt_key

# Google OAuth 2.0
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

# AI Services
GEMINI_API_KEY=your_gemini_api_key
ANTHROPIC_API_KEY=your_anthropic_api_key   # optional

# Frontend URL (for CORS)
CLIENT_URL=http://localhost:5500
```

### 3 — Install Dependencies & Start the Backend

```bash
cd server
npm install
npm run dev        # development (nodemon)
# or
npm start          # production
```

The API will be available at `http://localhost:4000`.

### 4 — Serve the Frontend

Open `index.html` directly with [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) (VS Code extension) on port **5500**, or any static file server of your choice.

```
http://localhost:5500
```

## 🚢 Deployment

### Frontend → Vercel

The frontend is configured via `vercel.json` and deploys automatically on push.

```bash
vercel --prod
```

### Backend → Railway

The backend is configured via `server/railway.json`. Connect your Railway project to the repository and set the environment variables in the Railway dashboard.
